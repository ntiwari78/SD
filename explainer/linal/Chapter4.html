4.1

Real Vector Spaces

A vector space is any set where addition and scalar multiplication satisfy ten axioms. This abstraction unifies arrows in $\mathbb{R}^3$, polynomials, matrices, and continuous functions under one roof.

Think of a vector space like a perfectly well-behaved "calculator society." Members can add each other and be scaled by numbers — as long as the result stays in the society and follows 10 social rules. Arrows in 3D follow these rules. So do polynomials. That's why the same theorems apply to both!

DEFINITION — Vector Space

A real vector space $V$ is a non-empty set with two operations — addition $\mathbf{u}+\mathbf{v}$ and scalar multiplication $c\mathbf{u}$ — satisfying the following 10 axioms for all $\mathbf{u},\mathbf{v},\mathbf{w}\in V$ and scalars $k,m\in\mathbb{R}$:

  1. Closure under addition: $\mathbf{u}+\mathbf{v}\in V$
  2. Commutativity: $\mathbf{u}+\mathbf{v}=\mathbf{v}+\mathbf{u}$
  3. Associativity of addition: $(\mathbf{u}+\mathbf{v})+\mathbf{w}=\mathbf{u}+(\mathbf{v}+\mathbf{w})$
  4. Zero vector: $\exists\,\mathbf{0}$ s.t. $\mathbf{u}+\mathbf{0}=\mathbf{u}$
  5. Additive inverse: $\exists\,{-\mathbf{u}}$ s.t. $\mathbf{u}+(-\mathbf{u})=\mathbf{0}$
  6. Closure under scalar mult: $k\mathbf{u}\in V$
  7. Distributivity I: $k(\mathbf{u}+\mathbf{v})=k\mathbf{u}+k\mathbf{v}$
  8. Distributivity II: $(k+m)\mathbf{u}=k\mathbf{u}+m\mathbf{u}$
  9. Associativity of scalar mult: $k(m\mathbf{u})=(km)\mathbf{u}$
  10. Unity: $1\cdot\mathbf{u}=\mathbf{u}$

4.2

Subspaces

A subspace is a subset of a vector space that is itself a vector space. Rather than checking all 10 axioms, we need only verify three conditions.

A subspace is a "room within a house." The house (full vector space) has all the furniture and rules. The room inherits all those rules automatically — you only need to confirm: does the room have a zero? Does adding two room-dwellers keep you in the room? Does scaling a room-dweller keep you in the room?

THEOREM — Subspace Test (3 Conditions)

A non-empty subset $W$ of a vector space $V$ is a subspace of $V$ if and only if:

  1. Zero vector: $\mathbf{0}\in W$
  2. Closed under addition: If $\mathbf{u},\mathbf{v}\in W$, then $\mathbf{u}+\mathbf{v}\in W$
  3. Closed under scalar multiplication: If $\mathbf{u}\in W$ and $c\in\mathbb{R}$, then $c\mathbf{u}\in W$

DEFINITION — Span

If $S=\{\mathbf{v}_1,\ldots,\mathbf{v}_r\}$ is a set of vectors in $V$, then the span of $S$ is the set of all linear combinations:

$\text{span}(S)=\{c_1\mathbf{v}_1+c_2\mathbf{v}_2+\cdots+c_r\mathbf{v}_r : c_i\in\mathbb{R}\}$

$\text{span}(S)$ is always a subspace of $V$ (it is the smallest subspace containing $S$).

A plane $ax+by+cz=d$ with $d\ne 0$ is NOT a subspace — it doesn't contain $\mathbf{0}$. Subspaces must always pass through the origin.
Affine vs. Linear Subspaces in ML. The solution set $A\mathbf{x}=\mathbf{b}$ (with $\mathbf{b}\ne\mathbf{0}$) is an affine subspace — a shifted linear subspace. Many ML models (SVMs, linear regression) find optimal separating hyperplanes which are affine subspaces.
4.3

Linear Independence

Vectors are linearly independent if none can be written as a combination of the others — they contribute genuinely new directions. This is the cornerstone of basis theory.

Independence means "no redundancy." If you can describe where you are with 2 steps (North, East), adding a "North-East" direction is redundant — it's just a combination of the first two. Linearly independent vectors are like a minimal, non-redundant GPS system.

DEFINITION — Linear Independence

Vectors $\mathbf{v}_1,\ldots,\mathbf{v}_n\in V$ are linearly independent if the only solution to

$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_n\mathbf{v}_n = \mathbf{0}$

is $c_1=c_2=\cdots=c_n=0$ (the trivial solution). Otherwise they are linearly dependent.

Are $\mathbf{v}_1=(1,2,3)$, $\mathbf{v}_2=(0,1,2)$, $\mathbf{v}_3=(2,3,4)$ linearly independent?
Form $c_1\mathbf{v}_1+c_2\mathbf{v}_2+c_3\mathbf{v}_3=\mathbf{0}$ and row-reduce the coefficient matrix:
$$\begin{bmatrix}1&0&2\\2&1&3\\3&2&4\end{bmatrix}\to\begin{bmatrix}1&0&2\\0&1&-1\\0&0&0\end{bmatrix}$$ Free variable $c_3=t$: $c_1=-2t$, $c_2=t$. Non-trivial solution exists (e.g. $t=1$): $-2\mathbf{v}_1+\mathbf{v}_2+\mathbf{v}_3=\mathbf{0}$. Dependent!

THEOREM — Dependence Criteria

  • A set containing $\mathbf{0}$ is always dependent.
  • A set of 1 vector is independent iff it is non-zero.
  • A set of 2 vectors is dependent iff one is a scalar multiple of the other.
  • Any set of $>n$ vectors in $\mathbb{R}^n$ is dependent.
  • $n$ vectors in $\mathbb{R}^n$ are independent iff their matrix has non-zero determinant.
Feature Redundancy. Linearly dependent features in a dataset are redundant — they add no new information. Principal Component Analysis (PCA) finds a maximally independent set of "directions" (principal components) capturing the variance. Detecting multicollinearity in regression is exactly testing for linear dependence.
4.4

Basis & Coordinates

A basis for a vector space is a linearly independent set that spans the space — the smallest possible spanning set, and the largest possible independent set.

DEFINITION — Basis

A set $S=\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ is a basis for $V$ if:

  1. $S$ is linearly independent, and
  2. $S$ spans $V$: every vector in $V$ is a linear combination of $S$.

DEFINITION — Coordinates

If $B=\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ is a basis for $V$ and $\mathbf{w}=c_1\mathbf{v}_1+\cdots+c_n\mathbf{v}_n$, then the scalars $c_1,\ldots,c_n$ are the coordinates of $\mathbf{w}$ relative to $B$, written:

$(\mathbf{w})_B = (c_1, c_2, \ldots, c_n)$

Uniqueness of Coordinates. If $B$ is a basis, every vector has exactly one representation in $B$. This is why bases are so powerful: they give a unique "address" to every point in the space. Change the basis, change the coordinates — but the vector itself is unchanged.
Encodings = Coordinates. When a word-embedding model maps "cat" to a 300-dimensional vector, those 300 numbers are the coordinates of "cat" in the embedding basis. The choice of basis (training the embedding) determines what geometric relationships like cosine similarity mean.
4.5

Dimension

The dimension of a vector space is the number of vectors in any basis. This is a fundamental invariant — every basis has the same size.

Dimension counts the minimum number of "independent questions" you need to answer to locate any point in the space. To find a point on a line: 1 question. On a plane: 2. In 3D space: 3. In Netflix's recommendation model: hundreds. Dimension is the irreducible complexity of the space.

THEOREM — Basis Size Invariance

All bases for a finite-dimensional vector space $V$ contain the same number of vectors. This common number is the dimension of $V$, denoted $\dim(V)$.

SpaceDimensionStandard Basis Size
$\mathbb{R}^n$$n$$n$
$P_n$$n+1$$n+1$

THEOREM — Plus/Minus Theorem

Let $V$ be an $n$-dimensional space, $S$ a subset of $V$:

  • If $S$ spans $V$ and $|S|=n$, then $S$ is a basis.
  • If $S$ is linearly independent and $|S|=n$, then $S$ is a basis.
  • Any spanning set can be reduced to a basis; any independent set can be extended to a basis.
Find $\dim(\text{span}\{(1,2,0),(2,4,0),(0,1,1)\})$.
Form a matrix with these as rows and row-reduce: $$\begin{bmatrix}1&2&0\\2&4&0\\0&1&1\end{bmatrix}\to\begin{bmatrix}1&2&0\\0&0&0\\0&1&1\end{bmatrix}\to\begin{bmatrix}1&2&0\\0&1&1\\0&0&0\end{bmatrix}$$ Two non-zero rows → two leading variables → $\dim=2$.

4.6

Change of Basis

When working in two different bases $B$ and $B'$, we need a transition matrix to convert coordinates between them. This is the linear algebra of re-labeling.

Imagine describing a city using GPS coordinates (latitude, longitude) vs. street addresses (block, avenue). Both systems describe the same locations. A change-of-basis matrix is the "translation table" between these two description systems.

DEFINITION — Transition Matrix

Let $B=\{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$ and $B'=\{\mathbf{b}'_1,\ldots,\mathbf{b}'_n\}$ be two bases for $V$. The transition matrix from $B$ to $B'$ is the $n\times n$ matrix $P_{B\to B'}$ whose $j$-th column is $(\mathbf{b}_j)_{B'}$ (coordinates of $\mathbf{b}_j$ in basis $B'$). Then:

$(\mathbf{w})_{B'} = P_{B\to B'}\,(\mathbf{w})_B$

Find $P_{B\to B'}$ where $B=\{(1,0),(0,1)\}$ (standard) and $B'=\{(1,1),(1,-1)\}$.
We need $(1,0)$ and $(0,1)$ expressed in $B'$. Solve $c_1(1,1)+c_2(1,-1)=(1,0)$:
$c_1+c_2=1,\;c_1-c_2=0 \Rightarrow c_1=c_2=\tfrac12$. So $(1,0)_{B'}=(\tfrac12,\tfrac12)$.
Similarly $(0,1)_{B'}=(\tfrac12,-\tfrac12)$. Therefore: $$P_{B\to B'}=\begin{bmatrix}1/2 & 1/2\\ 1/2 & -1/2\end{bmatrix}$$ Check: $P_{B\to B'}(3,1)^T=P_{B\to B'}$ gives $(2,-1)^T$, meaning $(3,1)=2(1,1)+(-1)(1,-1)$. ✓

THEOREM — Computing $P_{B\to B'}$

Form the partitioned matrix $[B' | B]$ (columns of $B'$ on left, columns of $B$ on right) and row-reduce to $[I | P_{B\to B'}]$. Equivalently:

$P_{B\to B'} = (P_{B'\to\text{std}})^{-1}\cdot P_{B\to\text{std}}$

where $P_{B\to\text{std}}$ is the matrix with $B$-vectors as columns.

Inverse Transition. $(P_{B\to B'})^{-1} = P_{B'\to B}$. To go back, just invert the transition matrix. And $P_{B\to B'}P_{B'\to B}=I$ — a beautiful consistency check.
Eigendecomposition = Change of Basis. When you diagonalize a matrix $A=PDP^{-1}$, you are changing to the eigenbasis (columns of $P$) where $A$ acts as a simple scaling $D$. This is exactly a change of basis to the "natural coordinates" of $A$ — the foundation of PCA, spectral clustering, and PageRank.
4.7

Row Space, Column Space & Null Space

Every matrix $A$ has three fundamental subspaces baked into it. Understanding them reveals everything about what $A$ does and doesn't do to vectors.

A matrix $A$ is like a machine: input goes in ($\mathbf{x}$), output comes out ($A\mathbf{x}$). The column space is all possible outputs. The null space is all inputs that get crushed to zero (blindspots). The row space is the "active input directions." Together they tell you: what can the machine produce, and what does it ignore?

DEFINITIONS — Three Fundamental Subspaces

For an $m\times n$ matrix $A$:

  • Row space $\text{row}(A)$: span of the row vectors of $A$ — a subspace of $\mathbb{R}^n$
  • Column space $\text{col}(A)$: span of the column vectors of $A$ — a subspace of $\mathbb{R}^m$
  • Null space $\text{null}(A)$: solution set of $A\mathbf{x}=\mathbf{0}$ — a subspace of $\mathbb{R}^n$
Finding a basis for row$(A)$: Row-reduce $A$ to row echelon form. The non-zero rows of the REF form a basis for row$(A)$.
Finding a basis for col$(A)$: Row-reduce $A$ and identify the pivot columns. The corresponding columns of the original $A$ form a basis for col$(A)$.
Finding a basis for null$(A)$: Solve $A\mathbf{x}=\mathbf{0}$ by row reduction. Express the free variables parametrically — the resulting vectors form a basis for null$(A)$.

For the column space, always return to the original matrix to pick pivot columns — not the row-reduced one! Row operations change the column space. Row operations preserve the row space and the null space but not the column space.
4.8

Rank, Nullity & the Rank-Nullity Theorem

The rank counts the "active dimensions" of a matrix, while the nullity counts its "crushed dimensions." Their sum is always $n$ — one of the most elegant theorems in all of linear algebra.

DEFINITIONS — Rank and Nullity

For an $m\times n$ matrix $A$:

  • $\text{rank}(A) = \dim(\text{row}(A)) = \dim(\text{col}(A)) =$ number of pivot columns
  • $\text{nullity}(A) = \dim(\text{null}(A)) =$ number of free variables

THEOREM — Rank-Nullity (Dimension Theorem)

$\text{rank}(A) + \text{nullity}(A) = n$

where $n$ is the number of columns of $A$. The active and crushed dimensions partition the full input space.

Intrinsic Dimension of Data. If you have $n$ features but the data matrix has rank $r \ll n$, the data actually lives in an $r$-dimensional subspace — intrinsic dimensionality. 4.9

Matrix Transformations from $\mathbb{R}^n$ to $\mathbb{R}^m$

A matrix $A$ defines a linear transformation $T_A:\mathbb{R}^n\to\mathbb{R}^m$. The kernel and range of $T_A$ are exactly the null space and column space of $A$.

A matrix transformation is a function $T(\mathbf{x})=A\mathbf{x}$ that "reshapes space." It takes vectors from $\mathbb{R}^n$ (input space) and maps them into $\mathbb{R}^m$ (output space). The kernel is the set of inputs mapped to zero (the "deaf spot"), and the range is all reachable outputs (the "voice").

DEFINITIONS — Kernel and Range

For $T:\mathbb{R}^n\to\mathbb{R}^m$, $T(\mathbf{x})=A\mathbf{x}$:

  • Kernel $\ker(T)=\{\mathbf{x}\in\mathbb{R}^n : T(\mathbf{x})=\mathbf{0}\}=\text{null}(A)$ — subspace of $\mathbb{R}^n$
  • Range $R(T)=\{T(\mathbf{x}) : \mathbf{x}\in\mathbb{R}^n\}=\text{col}(A)$ — subspace of $\mathbb{R}^m$
  • Nullity $=\dim(\ker T)$; Rank $=\dim(R(T))$

THEOREM — One-to-One and Onto

For $T_A:\mathbb{R}^n\to\mathbb{R}^m$:

  • $T$ is one-to-one (injective) $\Leftrightarrow$ $\ker(T)=\{\mathbf{0}\}$ $\Leftrightarrow$ $\text{nullity}(A)=0$ $\Leftrightarrow$ $\text{rank}(A)=n$
  • $T$ is onto (surjective) $\Leftrightarrow$ $R(T)=\mathbb{R}^m$ $\Leftrightarrow$ $\text{rank}(A)=m$
  • $T$ is invertible (bijective) $\Leftrightarrow$ $m=n$ and $A$ is invertible
Let $T:\mathbb{R}^3\to\mathbb{R}^2$, $T(x,y,z)=(x+y, y+z)$. Find kernel and range.
Matrix: $A=\begin{bmatrix}1&1&0\\0&1&1\end{bmatrix}$. Row reduce: already in REF, rank$=2$.
Kernel: $x+y=0,\,y+z=0\Rightarrow x=t,\,y=-t,\,z=t$. Kernel $=\{t(1,-1,1)\}$ — a line in $\mathbb{R}^3$. Nullity$=1$.
Range: rank$=2=m$, so $T$ is onto: Range $=\mathbb{R}^2$.
Check: $2+1=3=n$. ✓ $T$ is onto but not one-to-one (nullity $>0$).
The Four Fundamental Subspaces (Strang's Picture). Every $m\times n$ matrix $A$ has four fundamental subspaces: row$(A)$ and null$(A)$ partition $\mathbb{R}^n$ orthogonally; col$(A)$ and null$(A^T)$ partition $\mathbb{R}^m$ orthogonally. These "Big Picture" relationships are the deepest structure of linear maps.
Neural Networks as Compositions of Linear Maps. Each layer of a neural network is (before the activation) a linear map $T_W(\mathbf{x})=W\mathbf{x}+\mathbf{b}$. The kernel of $W$ is the "dead space" — inputs the layer cannot distinguish. Rank deficiency (when rank$(W)<\min(m,n)$) indicates redundant neurons, motivating network pruning and low-rank factorization techniques like LoRA.

10 questions covering vector spaces, subspaces, independence, basis, dimension, rank-nullity, and fundamental subspaces.

Q1. Which of the following is NOT one of the 10 vector space axioms?

B is correct. Vector spaces require an additive inverse (−u), not a multiplicative inverse (u⁻¹). That's the domain of fields, not vector spaces.

Q2. The set $W=\{(x,y,z)\in\mathbb{R}^3 : x+y=1\}$ is:

B is correct. The zero vector (0,0,0) satisfies 0+0=0≠1, so 0∉W. Without the zero vector, W cannot be a subspace.

Q3. The vectors $(1,0,1)$, $(0,1,1)$, $(1,1,2)$ are:

C is correct. v₃ = v₁+v₂ = (1,0,1)+(0,1,1) = (1,1,2). This non-trivial relation (c₁=c₂=1, c₃=−1) shows dependence.

Q4. The dimension of $P_3$ (polynomials of degree $\le 3$) is:

B is correct. The standard basis {1, x, x², x³} has 4 elements, so dim(P₃)=4. In general dim(Pₙ)=n+1.

Q5. A set $S$ in a vector space $V$ spans $V$ and contains exactly $\dim(V)$ vectors. Then $S$ is:

A is correct. By the Plus/Minus Theorem: a spanning set with exactly dim(V) vectors is automatically linearly independent — hence a basis.

Q6. For a $4\times 6$ matrix $A$ with rank 3, the nullity of $A$ is:

B is correct. Rank-Nullity: rank + nullity = n (# of columns) → 3 + nullity = 6 → nullity = 3.

Q7. The column space of $A$ equals $\mathbb{R}^m$ if and only if:

B is correct. col(A) = ℝᵐ ⟺ rank(A) = m ⟺ A has full row rank ⟺ Ax=b is consistent for every b.

Q8. For a linear transformation $T:\mathbb{R}^5\to\mathbb{R}^3$, the maximum possible rank of $T$ is:

B is correct. rank(T) = dim(range(T)) ≤ dim(ℝ³) = 3. The output space limits the rank.

Q9. The transition matrix $P_{B\to B'}$ satisfies:

B is correct. The transition matrix converts B-coordinates to B'-coordinates: (w)_{B'} = P_{B→B'} (w)_B. The inverse goes the other way.

Q10. The null space of $A$ and the row space of $A$ are:

B is correct. null(A) and row(A) are both subspaces of ℝⁿ, and they are orthogonal complements: every vector in ℝⁿ decomposes uniquely as a sum of one from each.
