Linear Transformations

Unify all of linear algebra under one roof. A linear map is the morphism of vector spaces — the structure-preserving function that connects abstract spaces to matrices, coordinates, and computation.

8.1

General Linear Transformations

A linear transformation $T:V\to W$ is a function between vector spaces that respects the two operations of addition and scalar multiplication. Every matrix-vector product you have ever computed is a special case of this universal idea.

Imagine you have a "machine" $T$ that processes vectors. The machine is linear if: (1) feeding in the sum of two vectors is the same as summing the outputs, and (2) scaling the input scales the output by the same factor. A linear machine never "warps" space in a nonlinear way — no shifting, no bending, no squishing origin away from origin.

DEFINITION — Linear Transformation

Let $V$ and $W$ be vector spaces over $\mathbb{R}$. A function $T:V\to W$ is a linear transformation (or linear map, or vector space homomorphism) if for all $\mathbf{u},\mathbf{v}\in V$ and all scalars $c$:

$$T(\mathbf{u}+\mathbf{v})=T(\mathbf{u})+T(\mathbf{v}) \quad \text{(additivity)}$$

$$T(c\mathbf{u})=cT(\mathbf{u}) \quad \text{(homogeneity)}$$

Equivalently, $T(c\mathbf{u}+d\mathbf{v})=cT(\mathbf{u})+dT(\mathbf{v})$ for all scalars $c,d$.

Zero preservation is automatic. Setting $c=0$ in homogeneity gives $T(\mathbf{0})=\mathbf{0}$. So any function that moves the origin is automatically non-linear — this quick check eliminates many candidates instantly.

The Kernel and Range

Two fundamental subspaces emerge from every linear map:

KERNEL & RANGE

For $T:V\to W$:

  • Kernel (null space): $\ker(T)=\{\mathbf{v}\in V : T(\mathbf{v})=\mathbf{0}\}$ — a subspace of $V$
  • Range (image): $\text{range}(T)=\{T(\mathbf{v}) : \mathbf{v}\in V\}$ — a subspace of $W$

Dimension Theorem: If $V$ is finite-dimensional,

$$\dim(\ker T)+\dim(\text{range}\,T)=\dim V$$

This is the rank-nullity theorem for abstract vector spaces.

Example — Differentiation as a Linear Map. Let $T:\mathcal{P}_n\to\mathcal{P}_{n-1}$ defined by $T(p)=p'$ (the derivative). Check linearity:
$T(p+q)=(p+q)'=p'+q'=T(p)+T(q)$ ✓
$T(cp)=(cp)'=cp'=cT(p)$ ✓
$\ker(T)=$ constant polynomials $\cong\mathbb{R}$ (dimension 1). $\text{range}(T)=\mathcal{P}_{n-1}$ (dimension $n$). Check: $1+n=n+1=\dim\mathcal{P}_n$ ✓

Injective, Surjective, Bijective

THEOREM — One-to-One & Onto Characterizations

Let $T:V\to W$ be a linear transformation.

  • $T$ is one-to-one (injective) $\iff$ $\ker(T)=\{\mathbf{0}\}$
  • $T$ is onto (surjective) $\iff$ $\text{range}(T)=W$
  • $T$ is bijective (one-to-one and onto) $\iff$ $T$ is an isomorphism

Neural Network Layers. Every linear layer in a neural network is a linear transformation $T(\mathbf{x})=W\mathbf{x}+\mathbf{b}$ — technically an affine map (not linear because of the bias $\mathbf{b}$). The linear part $W\mathbf{x}$ lives in our theory; activation functions like ReLU are what introduce true non-linearity. Understanding the kernel of each layer explains why networks can collapse information (dimensionality reduction).

8.2

Compositions and Inverse Transformations

Linear maps compose to give new linear maps, and bijective linear maps have inverse maps that are also linear. This makes the collection of all linear maps between two spaces itself a rich algebraic object.

Think of linear transformations as "lenses" that reshape space. Composing two lenses means looking through both sequentially: first $T$ reshapes, then $S$ reshapes the result. The inverse lens undoes the reshaping — but only if the original lens was a perfect one-to-one correspondence (no information lost).

COMPOSITION OF LINEAR MAPS

If $T:U\to V$ and $S:V\to W$ are linear, their composition $S\circ T:U\to W$ defined by

$$(S\circ T)(\mathbf{u})=S(T(\mathbf{u}))$$

is also a linear transformation. Furthermore, if $B'$ is the standard matrix of $T$ and $A'$ is the standard matrix of $S$, then the standard matrix of $S\circ T$ is the product $A'B'$.

This is precisely why matrix multiplication is defined the way it is!

Properties of Composition

THEOREM — Algebraic Properties

For composable linear maps $R$, $S$, $T$ and scalar $c$:

  • Associativity: $(R\circ S)\circ T=R\circ(S\circ T)$
  • Distributivity: $R\circ(S+T)=R\circ S+R\circ T$
  • Scalar: $c(S\circ T)=(cS)\circ T=S\circ(cT)$
  • Identity: $I_W\circ T=T=T\circ I_V$ where $I_V(\mathbf{v})=\mathbf{v}$

Note: Composition is generally not commutative: $S\circ T\neq T\circ S$ in general.

Inverse Transformations

INVERSE LINEAR TRANSFORMATION

If $T:V\to W$ is bijective (one-to-one and onto), then there exists a unique function $T^{-1}:W\to V$ such that:

$$T^{-1}\circ T=I_V \qquad \text{and} \qquad T\circ T^{-1}=I_W$$

$T^{-1}$ is called the inverse transformation, and it is automatically linear.

The Matrix Connection. If $T:\mathbb{R}^n\to\mathbb{R}^n$ has standard matrix $A$, then:
  • $T$ is one-to-one $\iff$ $A$ is invertible ($\det A\neq 0$)
  • $T$ is onto $\iff$ $A$ is invertible
  • $T^{-1}$ has standard matrix $A^{-1}$
For square matrices, one-to-one and onto are equivalent — another face of the Invertible Matrix Theorem.
Example — Rotation and Its Inverse. The rotation $T_\theta:\mathbb{R}^2\to\mathbb{R}^2$ by angle $\theta$ has matrix $R_\theta=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$. Its inverse is rotation by $-\theta$: $T_\theta^{-1}=T_{-\theta}$, with matrix $R_{-\theta}=R_\theta^T$ (since $R_\theta$ is orthogonal). Composition: $T_\alpha\circ T_\beta=T_{\alpha+\beta}$ — the angle addition formula is hidden in matrix multiplication!

If $S\circ T$ is one-to-one, must $T$ be one-to-one? Must $S$ be one-to-one? What about surjectivity?
Show Answer

$T$ must be injective (if $T(\mathbf{u})=T(\mathbf{v})$ then $S(T(\mathbf{u}))=S(T(\mathbf{v}))$ so $\mathbf{u}=\mathbf{v}$). But $S$ need not be injective! Example: $S:\mathbb{R}^3\to\mathbb{R}^2$, $T:\mathbb{R}^2\to\mathbb{R}^3$ injective, $S\circ T$ may be injective even if $S$ isn't. For surjectivity: if $S\circ T$ is onto, then $S$ must be onto, but $T$ need not be.

8.3

Isomorphism

Two vector spaces are isomorphic if there exists a bijective linear transformation between them. Isomorphic spaces are, from the linear algebra perspective, identical — same structure, different labels on the vectors.

An isomorphism is a "perfect dictionary" between two vector spaces. Every word in language $V$ has exactly one translation in language $W$, and the translations perfectly preserve all grammatical rules (linearity). If such a dictionary exists, the two languages are the same — you just call the vectors by different names.

ISOMORPHISM

A linear transformation $T:V\to W$ is an isomorphism if $T$ is both one-to-one and onto. We say $V$ and $W$ are isomorphic, written $V\cong W$.

Properties preserved under isomorphism (invariants):

  • Dimension
  • Linear independence/dependence of sets
  • Spanning properties
  • Basis cardinality

FUNDAMENTAL THEOREM OF ISOMORPHISMS

Every $n$-dimensional real vector space is isomorphic to $\mathbb{R}^n$.

Proof sketch: If $B=\{\mathbf{b}_1,\ldots,\mathbf{b}_n\}$ is a basis for $V$, define $T:V\to\mathbb{R}^n$ by $T(\mathbf{b}_i)=\mathbf{e}_i$ and extend linearly. This is the coordinate isomorphism — it maps each vector to its coordinate vector with respect to $B$.

Consequence: All $n$-dimensional real vector spaces are isomorphic to each other.

Why Matrices Suffice. This theorem explains why you can always work with coordinate vectors and matrices instead of abstract vectors and linear maps. The isomorphism $V\cong\mathbb{R}^n$ means: once you fix a basis, every abstract calculation reduces to matrix arithmetic. The choice of basis is arbitrary — like choosing a coordinate system in geometry.

The Space of Linear Maps

THEOREM — $\mathcal{L}(V,W)$ is a Vector Space

The set $\mathcal{L}(V,W)$ of all linear transformations from $V$ to $W$ forms a vector space under pointwise addition and scalar multiplication:

  • $(S+T)(\mathbf{v})=S(\mathbf{v})+T(\mathbf{v})$
  • $(cT)(\mathbf{v})=c\cdot T(\mathbf{v})$

If $\dim V=n$ and $\dim W=m$, then $\dim\mathcal{L}(V,W)=mn$, and $\mathcal{L}(V,W)\cong M_{m\times n}(\mathbb{R})$.

Example — $\mathcal{P}_2\cong\mathbb{R}^3$. The space of degree-$\leq 2$ polynomials has basis $\{1,x,x^2\}$, dimension 3. The map $T:a+bx+cx^2\mapsto(a,b,c)$ is an isomorphism to $\mathbb{R}^3$. Under this isomorphism:
  • Polynomial addition ↔ vector addition in $\mathbb{R}^3$
  • Scalar multiplication ↔ scalar multiplication in $\mathbb{R}^3$
  • Differentiation $D:\mathcal{P}_2\to\mathcal{P}_1$ becomes the matrix $\begin{pmatrix}0&1&0\\0&0&2\end{pmatrix}$

Is $\mathcal{P}_3$ (polynomials of degree ≤ 3) isomorphic to $M_{2\times 2}(\mathbb{R})$ (2×2 matrices)? Why or why not?
Show Answer

Yes! Both have dimension 4. $\mathcal{P}_3$ has basis $\{1,x,x^2,x^3\}$ and $M_{2\times 2}$ has basis of four elementary matrices $\{E_{11},E_{12},E_{21},E_{22}\}$. Any bijective linear map between them (e.g., map $1\mapsto E_{11}$, $x\mapsto E_{12}$, etc.) is an isomorphism. Since both are 4-dimensional real vector spaces, they are isomorphic to $\mathbb{R}^4$ and hence to each other.

8.4

Matrices for General Linear Transformations

Every linear transformation $T:V\to W$ between finite-dimensional spaces can be represented as a matrix — once you fix bases. The matrix of $T$ relative to the bases $B$ and $B'$ encodes how $T$ moves basis vectors, and from that, everything.

Imagine you're a translator working between French and German. You only need to know how to translate each French word in the dictionary — because any French sentence is just a combination of dictionary words. Similarly, a linear map is completely determined by where it sends each basis vector. Write those outputs as coordinate column vectors in $W$'s basis: stack them into a matrix. Done.

MATRIX OF A LINEAR MAP

Let $T:V\to W$, $B=\{\mathbf{v}_1,\ldots,\mathbf{v}_n\}$ a basis for $V$, $B'=\{\mathbf{w}_1,\ldots,\mathbf{w}_m\}$ a basis for $W$. The matrix of $T$ relative to $B$ and $B'$ is the $m\times n$ matrix

$$[T]_{B'}^{B}=\bigl[[T(\mathbf{v}_1)]_{B'} \mid [T(\mathbf{v}_2)]_{B'} \mid \cdots \mid [T(\mathbf{v}_n)]_{B'}\bigr]$$

where $[T(\mathbf{v}_j)]_{B'}$ is the coordinate vector of $T(\mathbf{v}_j)$ in the basis $B'$. Then for any $\mathbf{v}\in V$:

$$[T(\mathbf{v})]_{B'} = [T]_{B'}^{B}\,[\mathbf{v}]_B$$

Fix bases $B$ and $B'$. For $V$ (domain) choose $B$; for $W$ (codomain) choose $B'$.
Apply $T$ to each basis vector $\mathbf{v}_j\in B$ to get $T(\mathbf{v}_j)\in W$.
Express each $T(\mathbf{v}_j)$ in the $B'$-basis of $W$ to get the coordinate vector $[T(\mathbf{v}_j)]_{B'}$.
Stack as columns of the matrix $[T]_{B'}^{B}$.
Verify: $[T(\mathbf{v})]_{B'}=[T]_{B'}^{B}[\mathbf{v}]_B$ for any $\mathbf{v}$.
Example — Differentiation on $\mathcal{P}_2$. Let $T=D:\mathcal{P}_2\to\mathcal{P}_1$ (differentiation). Bases: $B=\{1,x,x^2\}$ for $\mathcal{P}_2$, $B'=\{1,x\}$ for $\mathcal{P}_1$.
  • $D(1)=0=0\cdot 1+0\cdot x$, so column 1 is $(0,0)^T$
  • $D(x)=1=1\cdot 1+0\cdot x$, so column 2 is $(1,0)^T$
  • $D(x^2)=2x=0\cdot 1+2\cdot x$, so column 3 is $(0,2)^T$

$$[D]_{B'}^B=\begin{pmatrix}0&1&0\\0&0&2\end{pmatrix}$$

Check: $D(3+5x+2x^2)=5+4x$. Matrix gives $\begin{pmatrix}0&1&0\\0&0&2\end{pmatrix}\begin{pmatrix}3\\5\\2\end{pmatrix}=\begin{pmatrix}5\\4\end{pmatrix}$ ✓

THEOREM — Composition Corresponds to Matrix Product

If $T:U\to V$ and $S:V\to W$ with bases $B$ (for $U$), $B'$ (for $V$), $B''$ (for $W$), then:

$$[S\circ T]_{B''}^B=[S]_{B''}^{B'}\cdot[T]_{B'}^B$$

The matrix product of the two representation matrices equals the representation matrix of the composition. This is the fundamental theorem connecting abstract linear algebra to matrix algebra.

Show Answer

$T(1)=x=0\cdot 1+1\cdot x+0\cdot x^2$, column 1: $(0,1,0)^T$. $T(x)=x^2=0\cdot 1+0\cdot x+1\cdot x^2$, column 2: $(0,0,1)^T$. So $[T]_{B'}^B=\begin{pmatrix}0&0\\1&0\\0&1\end{pmatrix}$. Check dimensions: $3\times 2$ matrix maps $\mathbb{R}^2\to\mathbb{R}^3$ ✓

8.5

Similarity

When we represent the same linear transformation $T:V\to V$ using two different bases, we get two different matrices — but they are similar. Similar matrices represent the same geometric transformation, just described in different coordinate systems.

Suppose you describe a rotation using coordinates aligned with the axes of your room. Your friend in the same room uses coordinates rotated 45°. You both get matrices for the same rotation, but the matrices look different. They are similar — connected by a "change-of-coordinate" matrix that translates between your systems. The rotation hasn't changed; only your language for it has.

SIMILAR MATRICES

Square matrices $A$ and $B$ (both $n\times n$) are similar, written $A\sim B$, if there exists an invertible matrix $P$ such that:

$$B = P^{-1}AP$$

$P$ is the transition matrix (change-of-basis matrix) from the old basis to the new basis. Similarity is an equivalence relation.

Invariants Under Similarity

Similar matrices represent the same linear transformation and therefore share all "intrinsic" properties of the transformation:

THEOREM — Similarity Invariants

If $A\sim B$ (i.e., $B=P^{-1}AP$), then $A$ and $B$ have the same:

  • Determinant: $\det(B)=\det(P^{-1}AP)=\det(A)$
  • Trace: $\text{tr}(B)=\text{tr}(A)$ (sum of diagonal entries)
  • Rank and Nullity
  • Characteristic polynomial: $\det(\lambda I-B)=\det(\lambda I-A)$
  • Eigenvalues (same multiset, with same algebraic multiplicities)
  • Minimal polynomial

⚠️ Similar matrices need NOT have the same eigenvectors (eigenvectors are basis-dependent).

Diagonalization Revisited. A matrix $A$ is diagonalizable exactly when it is similar to a diagonal matrix $D=P^{-1}AP$. The columns of $P$ are the eigenvectors of $A$, and the diagonal entries of $D$ are the eigenvalues. Similarity is the unifying concept: $A$ and $D$ are different representations of the same linear transformation, just in different bases (standard basis vs. eigenbasis).
Example — Same Transformation, Different Matrices. Let $T:\mathbb{R}^2\to\mathbb{R}^2$ be the projection onto the $x$-axis.
  • Standard basis: $A=\begin{pmatrix}1&0\\0&0\end{pmatrix}$
  • Basis $B=\{(1,1),(1,-1)\}$: matrix is $B_{\text{rep}}=\begin{pmatrix}1/2&1/2\\1/2&1/2\end{pmatrix}$ — same eigenvalues $\{0,1\}$, different look

Both matrices are similar: $B_{\text{rep}}=P^{-1}AP$ where $P=\begin{pmatrix}1&1\\1&-1\end{pmatrix}$.

PCA as Change of Basis. Principal Component Analysis finds an orthonormal basis (the eigenvectors of the covariance matrix $\Sigma$) such that the representation of $\Sigma$ in that basis is diagonal ($\Lambda = P^T\Sigma P$, where $P$ is orthogonal). This is similarity (in fact, orthogonal similarity) applied to the covariance structure. Similarity ≠ Row Equivalence. Row-equivalent matrices (same reduced row echelon form) have the same solution sets, but they represent different linear maps in general. Similar matrices represent the same linear map in different bases. These are completely distinct equivalence relations — don't confuse them!
Two matrices have the same characteristic polynomial. Are they necessarily similar?
Show Answer

No! A classic counterexample: $A=\begin{pmatrix}0&0\\0&0\end{pmatrix}$ and $B=\begin{pmatrix}0&1\\0&0\end{pmatrix}$. Both have characteristic polynomial $\lambda^2$. But $A$ is diagonalizable (it's already diagonal — it's just zero) while $B$ is not ($B\neq P^{-1}AP$ for any $P$ since $A=0$ and $B\neq 0$). This shows the characteristic polynomial doesn't fully classify similarity — you also need to examine the Jordan form (minimal polynomial, eigenspace dimensions).

Chapter 8 Quiz

Test your mastery of linear transformations, composition, isomorphism, matrix representations, and similarity.

1. Which of the following is a linear transformation from $\mathbb{R}^2$ to $\mathbb{R}^2$?
Option (c) is linear: both components are linear in $x,y$. Option (a) has a constant $+1$ (shifts origin), option (b) has a product $xy$ (nonlinear), option (d) has squares (nonlinear).
2. If $T:\mathbb{R}^5\to\mathbb{R}^3$ is linear and onto, what is $\dim(\ker T)$?
By rank-nullity: $\dim\ker(T)+\dim\text{range}(T)=5$. Since $T$ is onto, $\dim\text{range}(T)=3$. So $\dim\ker(T)=5-3=2$.
3. A linear transformation $T:V\to W$ is one-to-one if and only if:
$T$ is injective $\iff$ $\ker(T)=\{\mathbf{0}\}$. This is the fundamental characterization. Note: $\dim V=\dim W$ is necessary but not sufficient — the kernel must be trivial.
4. If $T:\mathbb{R}^n\to\mathbb{R}^n$ has standard matrix $A$, which property guarantees $T$ is an isomorphism?
$T$ is bijective (isomorphism) $\iff$ $A$ is invertible. Symmetry says nothing about bijectivity; distinct eigenvalues implies diagonalizability but not necessarily invertibility (could have $\lambda=0$); trace $\neq 0$ is insufficient.
5. The composition $(S\circ T)(\mathbf{x})=S(T(\mathbf{x}))$ with standard matrices $A$ for $T$ and $B$ for $S$ has standard matrix:
$BA$: apply $T$ first (matrix $A$), then $S$ (matrix $B$). The composition matrix is $BA$ — right-to-left convention. This is why matrix multiplication is not commutative: $AB\neq BA$ in general, reflecting that composition order matters.
6. Every $n$-dimensional real vector space is isomorphic to:
The Fundamental Theorem: every $n$-dimensional real vector space is isomorphic to $\mathbb{R}^n$. The isomorphism is given by the coordinate map relative to any fixed basis. $M_{n\times n}$ has dimension $n^2$ and $\mathcal{P}_{n+1}$ has dimension $n+2$, so these are generally different.
7. To find the matrix $[T]_{B'}^B$ of $T:V\to W$ relative to bases $B$ and $B'$, the $j$-th column is:
Column $j$ of $[T]_{B'}^B$ is $[T(\mathbf{b}_j)]_{B'}$ — the coordinate vector (in the $B'$-basis of $W$) of the image of the $j$-th $B$-basis vector. This is the definition of the matrix representation.
8. Matrices $A$ and $B$ are similar ($B=P^{-1}AP$). Which invariant do they share?
Similar matrices share eigenvalues, characteristic polynomial, determinant, trace, rank, and nullity. They do NOT generally share eigenvectors (those are basis-dependent) or diagonal entries or row echelon form.
9. The differentiation map $D:\mathcal{P}_3\to\mathcal{P}_3$ (sending $p\mapsto p'$). The dimension of its kernel is:
$\ker(D)=$ constant polynomials $\{a_0 : a_0\in\mathbb{R}\}$, which is 1-dimensional. By rank-nullity: $1+\dim\text{range}(D)=4$, so $\dim\text{range}(D)=3=\dim\mathcal{P}_2$ ✓ (the range is $\mathcal{P}_2$, polynomials of degree ≤ 2).
10. If $A$ is similar to a diagonal matrix $D$, then the columns of the transition matrix $P$ (where $D=P^{-1}AP$) are:
The columns of $P$ are eigenvectors of $A$. The diagonal entries of $D$ are the corresponding eigenvalues. This is precisely the diagonalization formula from Chapter 5: $A=PDP^{-1}$ where $P$'s columns are linearly independent eigenvectors.
