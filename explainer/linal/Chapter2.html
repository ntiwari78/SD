Determinants

The signed volume of transformation — and the gatekeeper of invertibility

The determinant collapses an entire matrix into a single number that encodes how the transformation scales area (or volume). Zero determinant = singular matrix = information destroyed. Non-zero = invertible = information preserved. Cramer's Rule turns this into an elegant closed-form solution for any linear system.

§ 2.1

Determinants by Cofactor Expansion

The determinant is a single number that captures the "essence" of a square matrix — specifically, the signed factor by which the transformation scales volumes. We build it recursively using minors and cofactors.

🎓 Feynman Lens

Two vectors in 2D span a parallelogram. The determinant is the signed area of that parallelogram. In 3D, three vectors span a parallelepiped, and the determinant is its signed volume. The sign tells you whether the transformation preserves or reverses orientation — like distinguishing a right hand from a left hand.

Definition — Determinant of a 2×2 Matrix
For $A = \begin{bmatrix}a&b\\c&d\end{bmatrix}$, the determinant is: $$\det(A) = |A| = ad - bc$$ Geometrically: if columns of $A$ are the vectors $\mathbf{u}=(a,c)$ and $\mathbf{v}=(b,d)$, then $|\det(A)|$ = area of the parallelogram they span.
Definition — Minor and Cofactor
For an $n\times n$ matrix $A$:
  • The minor $M_{ij}$ is the determinant of the $(n-1)\times(n-1)$ matrix obtained by deleting row $i$ and column $j$ from $A$.
  • The cofactor $C_{ij} = (-1)^{i+j}M_{ij}$. The sign pattern is a checkerboard: $\begin{bmatrix}+&-&+\\-&+&-\\+&-&+\end{bmatrix}$
Theorem — Cofactor Expansion
The determinant of an $n\times n$ matrix $A$ can be computed by expanding along any row $i$ or column $j$: $$\det(A) = a_{i1}C_{i1} + a_{i2}C_{i2} + \cdots + a_{in}C_{in} \quad \text{(row }i\text{ expansion)}$$ $$\det(A) = a_{1j}C_{1j} + a_{2j}C_{2j} + \cdots + a_{nj}C_{nj} \quad \text{(column }j\text{ expansion)}$$ Strategy: always expand along the row or column with the most zeros.
Worked Example3×3 Determinant by Cofactor Expansion

Find $\det(A)$ for $A = \begin{bmatrix}3&1&-4\\2&5&6\\1&4&8\end{bmatrix}$, expanding along row 1.

  1. $\det(A) = 3\cdot C_{11} + 1\cdot C_{12} + (-4)\cdot C_{13}$
  2. $C_{11} = (+1)\begin{vmatrix}5&6\\4&8\end{vmatrix} = 40-24 = 16$
  3. $C_{12} = (-1)\begin{vmatrix}2&6\\1&8\end{vmatrix} = -(16-6) = -10$
  4. $C_{13} = (+1)\begin{vmatrix}2&5\\1&4\end{vmatrix} = 8-5 = 3$
  5. $\det(A) = 3(16) + 1(-10) + (-4)(3) = 48 - 10 - 12 = \mathbf{26}$

🤔 Socratic Check

If you expand a determinant along a row of all zeros, what do you get? What does this tell you about the matrix geometrically?

Reveal answer

You get $\det(A) = 0$. Geometrically, the rows are linearly dependent — the matrix is singular, meaning the transformation "crushes" $n$-dimensional space into at most $(n-1)$-dimensional space. The parallelotope has zero volume.

§ 2.2

Evaluating Determinants by Row Reduction

Cofactor expansion becomes unwieldy for large matrices ($n!$ terms for $n\times n$). Row reduction reduces the matrix to triangular form in $O(n^3)$ steps — far more efficient. But each row operation changes the determinant in a controlled way we must track.

🎓 Feynman Lens

Think of the determinant as the area of a room. If you swap two walls, the room flips orientation (sign changes). If you double one wall's length, the area doubles. But if you "lean" one wall while keeping its base (shearing/adding a multiple of one row to another), the area doesn't change at all — the base×height formula is unaffected. These three rules are the three elementary row operations.

Theorem — Effect of Row Operations on det(A)
Row Operation Effect on det
$R_i \leftrightarrow R_j$ (swap two rows)$\det \to -\det$ (sign reverses)
$R_i \leftarrow k R_i$ (scale row by $k\neq0$)$\det \to k\cdot\det$ (multiplied by $k$)
$R_i \leftarrow R_i + c R_j$ (add multiple)$\det$ unchanged ✓

Triangular matrix shortcut: $\det$ of a triangular matrix = product of its diagonal entries.

Worked ExampleComputing det by Row Reduction

Evaluate $\det(A)$ for $A = \begin{bmatrix}0&1&5\\3&-6&9\\2&6&1\end{bmatrix}$.

  1. Swap $R_1 \leftrightarrow R_2$ (to get non-zero pivot): $\det \to -\det$
    $$\begin{bmatrix}3&-6&9\\0&1&5\\2&6&1\end{bmatrix}$$
  2. $R_3 \leftarrow R_3 - \frac{2}{3}R_1$ (no det change):
    $$\begin{bmatrix}3&-6&9\\0&1&5\\0&10&-5\end{bmatrix}$$
  3. $R_3 \leftarrow R_3 - 10R_2$ (no det change):
    $$\begin{bmatrix}3&-6&9\\0&1&5\\0&0&-55\end{bmatrix}$$
  4. Upper triangular: $\det(\text{current}) = 3 \cdot 1 \cdot (-55) = -165$.
  5. Accounting for the swap: $\det(A) = -(-165) = \mathbf{165}$.

💡 Key Insight — Triangular Det is Easy

Upper triangular matrices have $\det = d_{11} \cdot d_{22} \cdots d_{nn}$ — just multiply the diagonal. Row reduction turns any matrix triangular in $O(n^3)$ operations, each with a tracked effect on det. This makes determinant computation scalable, while cofactor expansion stays stuck at $O(n!)$.

📊 Data Science Bridge

In numerical linear algebra, LU decomposition ($A = LU$ with pivoting) computes both $\det(A)$ and a system solution simultaneously. The log-determinant $\log|\det A|$ is numerically preferred over det itself — it avoids overflow for large matrices. This appears in Gaussian process regression, where the log-likelihood involves $\log|\det K|$ for a covariance matrix $K$.

⚠ Common Mistake

When scaling a row $R_i \leftarrow kR_i$, det is multiplied by $k$. So to "factor out" $k$ from a row: $\det(A') = k \cdot \det(A)$. But $\det(kA) = k^n \det(A)$ for an $n\times n$ matrix (all $n$ rows scaled). These are different — don't confuse "scale one row" vs "scale the whole matrix."

§ 2.3

Properties of Determinants; Cramer's Rule

Determinants obey elegant multiplicative laws and give us Cramer's Rule — a beautiful closed-form solution for any invertible square system $Ax = b$. We also obtain the adjoint formula for $A^{-1}$.

🎓 Feynman Lens

If two transformations $A$ and $B$ scale volumes by factors $|\det A|$ and $|\det B|$, then their composition $AB$ scales volume by $|\det A| \cdot |\det B|$. The determinant is a "volume scaling meter" — and meters multiply when operations compose. This is why $\det(AB) = \det(A)\det(B)$.

Theorem — Fundamental Properties of Determinants
  • $\det(AB) = \det(A)\cdot\det(B)$  (multiplicative)
  • $\det(A^T) = \det(A)$  (transpose preserves det)
  • $\det(A^{-1}) = \dfrac{1}{\det(A)}$  (when $A$ invertible)
  • $\det(kA) = k^n\det(A)$ for $n\times n$ matrix $A$
  • $A$ is invertible $\Leftrightarrow$ $\det(A)\neq 0$
  • If $A$ has a zero row/column, or two equal rows/columns: $\det(A) = 0$
  • $\det(A^k) = [\det(A)]^k$  (follows from multiplicativity)
Cramer's Rule
Let $A$ be an invertible $n\times n$ matrix and $b \in \mathbb{R}^n$. The unique solution $x = (x_1,\ldots,x_n)$ to $Ax = b$ is: $$x_j = \frac{\det(A_j)}{\det(A)}, \quad j = 1, 2, \ldots, n$$ where $A_j$ is the matrix obtained by replacing column $j$ of $A$ with $b$.

Adjoint formula for $A^{-1}$: $\displaystyle A^{-1} = \frac{1}{\det(A)}\,\text{adj}(A)$, where $\text{adj}(A)_{ij} = C_{ji}$ (transpose of cofactor matrix).
Worked ExampleSolving a 2×2 System with Cramer's Rule

Solve: $2x_1 + x_2 = 7$ and $5x_1 + 3x_2 = 17$.

  1. $A = \begin{bmatrix}2&1\\5&3\end{bmatrix}$, $b = \begin{bmatrix}7\\17\end{bmatrix}$. $\det(A) = 6-5 = 1$.
  2. $A_1 = \begin{bmatrix}7&1\\17&3\end{bmatrix}$: $\det(A_1) = 21-17 = 4$. So $x_1 = 4/1 = 4$.
  3. $A_2 = \begin{bmatrix}2&7\\5&17\end{bmatrix}$: $\det(A_2) = 34-35 = -1$. So $x_2 = -1/1 = -1$.
  4. Verify: $2(4)+(-1)=7$ ✓ and $5(4)+3(-1)=17$ ✓.
Worked ExampleFinding A⁻¹ via the Adjoint Formula

Find $A^{-1}$ using the adjoint for $A = \begin{bmatrix}2&1\\5&3\end{bmatrix}$.

  1. $\det(A) = 1$.
  2. Cofactors: $C_{11}=3$, $C_{12}=-5$, $C_{21}=-1$, $C_{22}=2$.
  3. $\text{adj}(A) = \begin{bmatrix}C_{11}&C_{21}\\C_{12}&C_{22}\end{bmatrix} = \begin{bmatrix}3&-1\\-5&2\end{bmatrix}$ (transpose of cofactor matrix).
  4. $A^{-1} = \dfrac{1}{1}\begin{bmatrix}3&-1\\-5&2\end{bmatrix}$. Verify: $AA^{-1} = I$ ✓.

💡 When to Use Cramer's Rule?

Cramer's Rule is elegant theoretically but computationally expensive: $O(n!)$ naive vs $O(n^3)$ for Gaussian elimination. Its power is theoretical — it gives explicit formulas for each variable as a ratio of determinants, which is invaluable for symbolic computation, sensitivity analysis, and proofs. In practice for $n \geq 4$, use Gaussian elimination.

📊 Data Science Bridge

The determinant inequality $|\det A| \leq \prod_i \|a_i\|$ (Hadamard's bound) underlies lattice-based cryptography. In statistics, Fisher's information matrix $\mathcal{I}(\theta)$ appears in the Cramér–Rao bound: $\text{Var}(\hat\theta) \geq \mathcal{I}^{-1}$. Its determinant measures how much data "constrains" the parameter — a geometric generalisation of variance. Log-det appears in multivariate Gaussian entropy: $H = \frac{1}{2}\log[(2\pi e)^n \det\Sigma]$.

🤔 Socratic Check

If $\det(A) = 5$ and $\det(B) = 3$, what is $\det(2AB^{-1}A^T)$ for $4\times 4$ matrices $A, B$?

Reveal answer

$\det(2AB^{-1}A^T) = 2^4 \cdot \det(A)\cdot\det(B)^{-1}\cdot\det(A^T) = 16 \cdot 5 \cdot \tfrac{1}{3} \cdot 5 = \dfrac{400}{3}$.
