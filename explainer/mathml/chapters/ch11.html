Topology, Limits, and Continuity

Open and closed sets, bounded and compact sets, limits, and continuity — the foundations of analysis.

40 min read Part II · Calculus
Mental Model

The Neighbourhood Test

Imagine you are standing at a point \(\mathbf{x}\) in a landscape. If you can always take a tiny step in any direction and stay inside your set, you are at an interior point — and a set made entirely of such points is open. If a set also contains every point on its boundary fence, it is closed. These seemingly simple ideas control when limits exist, when functions behave smoothly, and ultimately when gradient descent can find a guaranteed minimum.

Section 11.1

Open Sets

In \(\mathbb{R}^n\), a set is defined entirely by what happens in tiny neighbourhoods of each of its points. An open ball of radius \(\varepsilon > 0\) centred at \(\mathbf{x}\) is the set

\[B(\mathbf{x}, \varepsilon) = \{ \mathbf{y} \in \mathbb{R}^n : \|\mathbf{y} - \mathbf{x}\| < \varepsilon \}.\]

For \(n=1\) this is an open interval \((x - \varepsilon, \, x + \varepsilon)\); for \(n=2\) it is an open disk.

Definition — Open Set

A set \(U \subseteq \mathbb{R}^n\) is open if for every \(\mathbf{x} \in U\) there exists \(\varepsilon > 0\) such that \(B(\mathbf{x}, \varepsilon) \subseteq U\).

In other words, every point of an open set has "room to breathe" — you can always wiggle a little and stay inside.

Proposition — Properties of Open Sets

1. \(\mathbb{R}^n\) and \(\varnothing\) are both open.
2. Any union of open sets (even uncountably many) is open.
3. Any finite intersection of open sets is open.

The finiteness in property 3 is essential. The intersection of the infinitely many open intervals \((-1/n,\, 1/n)\) for \(n=1,2,3,\ldots\) is just \(\{0\}\), which is not open.

Interior Points

Given any set \(S \subseteq \mathbb{R}^n\), a point \(\mathbf{x} \in S\) is an interior point if some open ball around \(\mathbf{x}\) fits entirely in \(S\). The collection of all interior points is called the interior of \(S\), written \(\mathrm{int}(S)\) or \(S^\circ\). A set is open if and only if \(S = \mathrm{int}(S)\).

Example

The closed interval \([0, 1]\) is not open because 0 has no open ball \(B(0, \varepsilon)\) contained entirely in \([0, 1]\). Its interior is the open interval \((0, 1)\).

Section 11.2

Closed Sets

Definition — Closed Set

A set \(C \subseteq \mathbb{R}^n\) is closed if its complement \(\mathbb{R}^n \setminus C\) is open.

"Open" and "closed" are not opposites. A set can be both (these are called clopen): \(\mathbb{R}^n\) and \(\varnothing\) are both open and closed. A set can also be neither: the half-open interval \([0, 1)\) is neither open nor closed.

Limit Points and Closure

A point \(\mathbf{x}\) is a limit point (or accumulation point) of \(S\) if every open ball \(B(\mathbf{x}, \varepsilon)\) contains at least one point of \(S\) other than \(\mathbf{x}\) itself. The closure \(\overline{S}\) is the union of \(S\) with all its limit points.

Theorem — Sequential Characterisation of Closed Sets

A set \(C \subseteq \mathbb{R}^n\) is closed if and only if every convergent sequence \((\mathbf{x}_k)_{k=1}^\infty\) with \(\mathbf{x}_k \in C\) has its limit in \(C\).

(\(\Rightarrow\)) Suppose \(C\) is closed and \((\mathbf{x}_k) \to \mathbf{x}\) with each \(\mathbf{x}_k \in C\). If \(\mathbf{x} \notin C\), then \(\mathbf{x} \in \mathbb{R}^n \setminus C\), which is open. So there exists \(\varepsilon > 0\) with \(B(\mathbf{x}, \varepsilon) \subseteq \mathbb{R}^n \setminus C\). But convergence gives some \(N\) with \(\|\mathbf{x}_k - \mathbf{x}\| < \varepsilon\) for all \(k \geq N\), so \(\mathbf{x}_N \in B(\mathbf{x}, \varepsilon) \subseteq \mathbb{R}^n \setminus C\). This contradicts \(\mathbf{x}_N \in C\). Hence \(\mathbf{x} \in C\).

(\(\Leftarrow\)) Suppose every convergent sequence in \(C\) has its limit in \(C\). We show \(\mathbb{R}^n \setminus C\) is open. Take any \(\mathbf{x} \notin C\). If no \(\varepsilon > 0\) gives \(B(\mathbf{x}, \varepsilon) \subseteq \mathbb{R}^n \setminus C\), then for each \(k \in \mathbb{N}\) there is \(\mathbf{x}_k \in B(\mathbf{x}, 1/k) \cap C\). Then \(\mathbf{x}_k \to \mathbf{x}\), and by hypothesis \(\mathbf{x} \in C\) — contradiction. So the complement is open, hence \(C\) is closed. \(\blacksquare\)

This sequential characterisation is what practitioners use most: a closed set "keeps all its limit points." In optimization, the fact that a feasible set is closed guarantees that a convergent sequence of feasible solutions has a feasible limit.

Proposition — Properties of Closed Sets

1. \(\mathbb{R}^n\) and \(\varnothing\) are both closed.
2. Any intersection of closed sets (even uncountably many) is closed.
3. Any finite union of closed sets is closed.

Section 11.3

Bounded and Compact Sets

Definition — Bounded Set

A set \(S \subseteq \mathbb{R}^n\) is bounded if there exists \(M > 0\) such that \(\|\mathbf{x}\| \leq M\) for every \(\mathbf{x} \in S\). Equivalently, \(S\) fits inside some ball \(B(\mathbf{0}, M)\).

The open interval \((0, 1)\) is bounded. The set \(\mathbb{R}\) is not. Boundedness alone does not guarantee that optimization problems have solutions — we also need closedness.

Definition — Compact Set

A set \(K \subseteq \mathbb{R}^n\) is compact if it is both closed and bounded.

Theorem — Heine–Borel

A subset of \(\mathbb{R}^n\) is compact if and only if every open cover has a finite sub-cover.

The deep consequence: compact sets are exactly where analysis works "as well as it can." Every sequence in a compact set has a convergent subsequence (Bolzano–Weierstrass), and continuous functions on compact sets are guaranteed to attain their maximum and minimum.

Theorem — Bolzano–Weierstrass

Every bounded sequence in \(\mathbb{R}^n\) has a convergent subsequence. Equivalently, every sequence in a compact set \(K\) has a subsequence converging to a point in \(K\).

Why Compactness Matters for ML

Consider minimising a loss \(L(\boldsymbol\theta)\) over parameters \(\boldsymbol\theta \in K\). If \(L\) is continuous and \(K\) is compact, the Extreme Value Theorem guarantees a minimiser exists. Without compactness, the infimum might be approached but never achieved — like trying to minimise \(e^{-x}\) over \(\mathbb{R}\): the infimum is 0 but no \(x\) achieves it.

Compact Sets in Practice

In machine learning, we often enforce compactness through explicit constraints: weight clipping in WGANs constrains weights to \([-c, c]^d\), a compact set. Projected gradient descent clips iterates back onto a compact feasible region. Spectral normalisation bounds operator norms, implicitly keeping weight matrices in a compact set.

Python
import numpy as np

def project_onto_ball(theta, radius):
    """Project parameter vector onto a closed ball (compact set)."""
    norm = np.linalg.norm(theta)
    if norm > radius:
        return theta * (radius / norm)
    return theta

# Projected gradient descent on a compact feasible set
theta = np.random.randn(5)
lr, R = 0.01, 2.0
for step in range(1000):
    grad = compute_gradient(theta)         # your loss gradient
    theta = theta - lr * grad              # gradient step
    theta = project_onto_ball(theta, R)    # project back to compact set
Section 11.4

Limits of Functions

With the topology of open and closed sets in hand, we can rigorously define what it means for a function to approach a value.

Definition — Limit of a Function (ε–δ)

Let \(f : D \to \mathbb{R}^m\) where \(D \subseteq \mathbb{R}^n\), and let \(\mathbf{a}\) be a limit point of \(D\). We write

\[\lim_{\mathbf{x} \to \mathbf{a}} f(\mathbf{x}) = \mathbf{L}\]

if for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that \(0 < \|\mathbf{x} - \mathbf{a}\| < \delta\) and \(\mathbf{x} \in D\) imply \(\|f(\mathbf{x}) - \mathbf{L}\| < \varepsilon\).

The structure mirrors the ε-N definition for sequences (Chapter 10), but now we control spatial proximity through \(\delta\) rather than index proximity through \(N\).

Sequential Characterisation of Limits

Theorem — Sequential Criterion for Limits

\(\displaystyle\lim_{\mathbf{x} \to \mathbf{a}} f(\mathbf{x}) = \mathbf{L}\) if and only if for every sequence \((\mathbf{x}_k)\) in \(D \setminus \{\mathbf{a}\}\) with \(\mathbf{x}_k \to \mathbf{a}\), we have \(f(\mathbf{x}_k) \to \mathbf{L}\).

This is powerful for proving that limits do not exist: find two sequences approaching \(\mathbf{a}\) along different paths that produce different function values.

Example — Limit Does Not Exist

Consider \(f(x, y) = \frac{xy}{x^2 + y^2}\) at the origin. Along the path \(y = 0\), we get \(f(x, 0) = 0 \to 0\). Along \(y = x\), we get \(f(x, x) = x^2/(2x^2) = 1/2\). Since the two paths give different limits, \(\lim_{(x,y) \to (0,0)} f(x,y)\) does not exist.

Algebraic Rules for Limits

If \(\lim_{\mathbf{x}\to\mathbf{a}} f(\mathbf{x}) = L\) and \(\lim_{\mathbf{x}\to\mathbf{a}} g(\mathbf{x}) = M\), then the sum rule \(\lim(f+g) = L + M\), product rule \(\lim(fg) = LM\), and quotient rule \(\lim(f/g) = L/M\) (when \(M \neq 0\)) all hold. These carry over directly from the sequence algebra of Chapter 10 via the sequential criterion.

Python
import numpy as np

def explore_limit_2d(f, a, num_paths=8, num_points=50):
    """Probe a 2D limit by approaching along multiple straight-line paths."""
    a = np.asarray(a, dtype=float)
    angles = np.linspace(0, np.pi, num_paths, endpoint=False)
    results = {}
    for theta in angles:
        direction = np.array([np.cos(theta), np.sin(theta)])
        ts = np.logspace(-1, -8, num_points)
        vals = [f(*(a + t * direction)) for t in ts]
        results[f"θ={theta:.2f}"] = vals[-1]
    return results

# Test: f(x,y) = xy / (x² + y²)
f = lambda x, y: x*y / (x**2 + y**2) if x**2 + y**2 > 0 else 0
paths = explore_limit_2d(f, (0, 0))
for path, val in paths.items():
    print(f"{path}: {val:.6f}")
# Different paths → different values ⇒ limit does not exist
Section 11.5

Continuity

Continuity formalises the intuition that "small changes in input produce small changes in output" — the single most important smoothness property in calculus and machine learning.

Definition — Continuity at a Point (ε–δ)

A function \(f : D \to \mathbb{R}^m\) is continuous at \(\mathbf{a} \in D\) if for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that \(\|\mathbf{x} - \mathbf{a}\| < \delta\) and \(\mathbf{x} \in D\) imply \(\|f(\mathbf{x}) - f(\mathbf{a})\| < \varepsilon\).

Equivalently, \(f\) is continuous at \(\mathbf{a}\) if and only if \(\lim_{\mathbf{x}\to\mathbf{a}} f(\mathbf{x}) = f(\mathbf{a})\). Notice three things must hold simultaneously: the limit exists, \(f(\mathbf{a})\) is defined, and they are equal.

Definition — Continuous Function

\(f\) is continuous (on its entire domain) if it is continuous at every point \(\mathbf{a} \in D\).

Topological Characterisation

Theorem — Topological Continuity

A function \(f : D \to \mathbb{R}^m\) is continuous if and only if the preimage of every open set is open (relative to \(D\)). That is, for every open \(V \subseteq \mathbb{R}^m\), the set \(f^{-1}(V) = \{ \mathbf{x} \in D : f(\mathbf{x}) \in V \}\) is open in \(D\).

(\(\Rightarrow\)) Suppose \(f\) is continuous (ε-δ sense) and \(V \subseteq \mathbb{R}^m\) is open. Take any \(\mathbf{a} \in f^{-1}(V)\). Then \(f(\mathbf{a}) \in V\). Since \(V\) is open, there exists \(\varepsilon > 0\) with \(B(f(\mathbf{a}), \varepsilon) \subseteq V\). By continuity of \(f\) at \(\mathbf{a}\), there exists \(\delta > 0\) with \(f(B(\mathbf{a}, \delta) \cap D) \subseteq B(f(\mathbf{a}), \varepsilon) \subseteq V\). So \(B(\mathbf{a}, \delta) \cap D \subseteq f^{-1}(V)\), making \(f^{-1}(V)\) open in \(D\).

(\(\Leftarrow\)) Suppose preimages of open sets are open. Given \(\mathbf{a} \in D\) and \(\varepsilon > 0\), the ball \(V = B(f(\mathbf{a}), \varepsilon)\) is open. By hypothesis, \(f^{-1}(V)\) is open in \(D\) and contains \(\mathbf{a}\). So there exists \(\delta > 0\) with \(B(\mathbf{a}, \delta) \cap D \subseteq f^{-1}(V)\). This means \(\|\mathbf{x} - \mathbf{a}\| < \delta\) implies \(\|f(\mathbf{x}) - f(\mathbf{a})\| < \varepsilon\), which is ε-δ continuity at \(\mathbf{a}\). \(\blacksquare\)

This topological view is elegant and powerful: continuity is entirely about preserving the structure of open sets. It generalises seamlessly to abstract topological spaces and is the definition used in modern functional analysis.

Uniform Continuity

Definition — Uniform Continuity

\(f : D \to \mathbb{R}^m\) is uniformly continuous if for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that for all \(\mathbf{x}, \mathbf{y} \in D\), \(\|\mathbf{x} - \mathbf{y}\| < \delta\) implies \(\|f(\mathbf{x}) - f(\mathbf{y})\| < \varepsilon\).

The key difference: in ordinary continuity, \(\delta\) may depend on the point \(\mathbf{a}\). In uniform continuity, a single \(\delta\) works everywhere. The function \(f(x) = x^2\) on \(\mathbb{R}\) is continuous but not uniformly continuous (far from the origin, small \(\delta\) changes produce large output swings). On a compact domain \([0, M]\), however, every continuous function is automatically uniformly continuous.

Section 11.6

Properties of Continuous Functions

Theorem — Extreme Value Theorem

If \(f : K \to \mathbb{R}\) is continuous and \(K \subseteq \mathbb{R}^n\) is compact, then \(f\) attains its maximum and minimum on \(K\). That is, there exist \(\mathbf{x}^*, \mathbf{x}_* \in K\) such that \(f(\mathbf{x}_*) \leq f(\mathbf{x}) \leq f(\mathbf{x}^*)\) for all \(\mathbf{x} \in K\).

Let \(M = \sup_{\mathbf{x} \in K} f(\mathbf{x})\). There exists a sequence \((\mathbf{x}_k)\) in \(K\) with \(f(\mathbf{x}_k) \to M\). Since \(K\) is compact (Bolzano–Weierstrass), \((\mathbf{x}_k)\) has a convergent subsequence \(\mathbf{x}_{k_j} \to \mathbf{x}^* \in K\). By continuity, \(f(\mathbf{x}^*) = \lim f(\mathbf{x}_{k_j}) = M\). So the maximum is attained at \(\mathbf{x}^*\). The argument for the minimum is identical with inf. \(\blacksquare\)

This theorem is the theoretical foundation of constrained optimisation in ML. Whenever you minimise a continuous loss over a compact parameter set, a solution is guaranteed to exist.

Theorem — Intermediate Value Theorem

If \(f : [a, b] \to \mathbb{R}\) is continuous and \(f(a) \leq y \leq f(b)\) (or \(f(b) \leq y \leq f(a)\)), then there exists \(c \in [a, b]\) with \(f(c) = y\).

Continuous functions on connected domains cannot "jump" — they must pass through every intermediate value. This is why binary search (bisection) works for root-finding: if a continuous function is negative at one end and positive at the other, it must cross zero somewhere in between.

Composition Preserves Continuity

Proposition — Composition of Continuous Functions

If \(f\) is continuous at \(\mathbf{a}\) and \(g\) is continuous at \(f(\mathbf{a})\), then \(g \circ f\) is continuous at \(\mathbf{a}\).

This is why deep neural networks produce continuous functions (when using continuous activations like ReLU, sigmoid, or tanh): each layer is a continuous function (affine map composed with continuous activation), and the composition of finitely many continuous functions is continuous.

Lipschitz Continuity

Definition — Lipschitz Continuity

A function \(f : D \to \mathbb{R}^m\) is Lipschitz continuous with constant \(L \geq 0\) if

\[\|f(\mathbf{x}) - f(\mathbf{y})\| \leq L \|\mathbf{x} - \mathbf{y}\|\]

for all \(\mathbf{x}, \mathbf{y} \in D\).

Lipschitz continuity is stronger than uniform continuity (take \(\delta = \varepsilon / L\)). The Lipschitz constant \(L\) bounds how fast the function can change — it is a global "speed limit." In ML, Lipschitz constraints appear everywhere: spectral normalisation enforces \(L = 1\) on discriminator layers in GANs, and Lipschitz bounds on gradients are used to prove convergence rates for gradient descent.

Python
import numpy as np

def estimate_lipschitz(f, domain_samples, n_pairs=10000):
    """Estimate Lipschitz constant by sampling pairwise ratios."""
    n = len(domain_samples)
    max_ratio = 0.0
    for _ in range(n_pairs):
        i, j = np.random.choice(n, 2, replace=False)
        x_diff = np.linalg.norm(domain_samples[i] - domain_samples[j])
        if x_diff < 1e-12:
            continue
        f_diff = np.linalg.norm(f(domain_samples[i]) - f(domain_samples[j]))
        max_ratio = max(max_ratio, f_diff / x_diff)
    return max_ratio

# ReLU: Lipschitz constant = 1
relu = lambda x: np.maximum(x, 0)
samples = np.random.randn(5000, 1)
L_hat = estimate_lipschitz(
    lambda x: relu(x), samples
)
print(f"Estimated Lipschitz constant of ReLU: {L_hat:.4f}")  # ≈ 1.0
Section 11.7

Topology in ML

Why This Matters for AI/ML

From Theory to Practice

  • Guaranteed optima: The Extreme Value Theorem tells us that minimising a continuous loss over a compact feasible set always has a solution. Weight clipping, projected gradient descent, and norm constraints all create compact sets for this reason.
  • Stability of predictions: Lipschitz continuity bounds output perturbation relative to input perturbation. An \(L\)-Lipschitz classifier changes its output by at most \(L\varepsilon\) when inputs are perturbed by \(\varepsilon\) — the mathematical basis of robustness certificates.
  • GAN training: The Wasserstein GAN requires the discriminator (critic) to be 1-Lipschitz. Spectral normalisation and gradient penalty enforce this constraint, with the topological guarantee that continuous bounded functions on compact sets are well-behaved.
  • Network architectures: Activation functions (ReLU, sigmoid, tanh, softmax) are all continuous, so the composition of layers with these activations produces a continuous function. This is required for the chain rule (Chapter 12) and backpropagation to work.
  • Convergence proofs: Nearly every convergence proof for optimisation algorithms assumes continuity (often Lipschitz continuity of the gradient). Without these topological properties, there are no convergence guarantees.

The Hierarchy of Smoothness

The concepts in this chapter form a hierarchy of increasingly strong regularity conditions that appear throughout ML theory:

\[\text{Continuous} \;\subset\; \text{Uniformly continuous} \;\subset\; \text{Lipschitz continuous} \;\subset\; \text{Differentiable}\]

Each level adds guarantees. Continuity ensures limits exist. Uniform continuity ensures uniform approximation. Lipschitz ensures bounded rate of change. Differentiability (Chapter 12) gives a local linear approximation. Stronger assumptions yield stronger convergence rates for gradient-based optimisers.

Performance Deep-Dive

Checking Continuity: Numerical ε-δ Testing

While continuity is a theoretical property, we can numerically probe it — especially useful for debugging custom activation functions or verifying that model outputs are continuous.

import numpy as np

def check_continuity(f, points, deltas=None):
    """Numerically check ε-δ continuity at given points."""
    if deltas is None:
        deltas = [1e-2, 1e-4, 1e-6, 1e-8]
    results = []
    for x0 in points:
        f0 = f(x0)
        ratios = []
        for delta in deltas:
            # Sample random perturbations of size ≤ delta
            perturbs = x0 + delta * (2 * np.random.rand(100) - 1)
            max_eps = max(abs(f(p) - f0) for p in perturbs)
            ratios.append(max_eps)
        # If function is continuous, max_eps → 0 as delta → 0
        converges = all(ratios[i] >= ratios[i+1] * 0.5
                        for i in range(len(ratios)-1))
        results.append({"point": x0, "deltas": deltas,
                        "max_eps": ratios, "likely_continuous": converges})
    return results

# Test: ReLU is continuous everywhere (including at 0)
relu = lambda x: max(0, x)
report = check_continuity(relu, [-1.0, 0.0, 0.5, 1.0])
for r in report:
    status = "✓" if r["likely_continuous"] else "✗"
    print(f"x={r['point']:5.1f}: max|f(x')-f(x)| → "
          f"{r['max_eps'][-1]:.2e}  {status}")

Socratic Knowledge Check

You are designing a neural network for a safety-critical system. The network must produce outputs that change "smoothly" with inputs. Your colleague asks: "Is it enough that our network is continuous, or do we need Lipschitz continuity?"
1
Is the set \(\{(x, y) \in \mathbb{R}^2 : x^2 + y^2 < 1\}\) open, closed, both, or neither? What about \(\{(x, y) : x^2 + y^2 \leq 1\}\)?
The open disk \(x^2 + y^2 < 1\) is open: every point inside has a small ball that fits entirely within the disk. The closed disk \(x^2 + y^2 \leq 1\) is closed: it contains all its boundary points. The open disk is bounded and open (not compact). The closed disk is bounded and closed, hence compact.
2
Explain why \(f(x) = 1/x\) is continuous on \((0, \infty)\) but not uniformly continuous. What topological property does \((0, \infty)\) lack?
At any \(a > 0\), given \(\varepsilon\), we can find \(\delta\) depending on \(a\). But near 0, the function blows up: for points \(x = 1/n, y = 1/(n+1)\), \(|x - y| \to 0\) while \(|f(x) - f(y)| = n(n+1)/(n+1) - n \to 1\). No single \(\delta\) works for all points. The domain \((0, \infty)\) is not compact — it is not bounded. On any compact sub-interval \([a, b]\) with \(a > 0\), the function is uniformly continuous.
3
Your network uses ReLU activations. Is the overall network function Lipschitz continuous? What determines the Lipschitz constant?
Yes. ReLU is 1-Lipschitz, and an affine map \(\mathbf{x} \mapsto W\mathbf{x} + \mathbf{b}\) is \(\|W\|\)-Lipschitz (operator norm). By composition, a depth-\(d\) network has Lipschitz constant at most \(\prod_{i=1}^{d} \|W_i\|\). The product of the weight matrix norms is the key quantity. Spectral normalisation divides each \(W_i\) by its largest singular value, ensuring each layer is 1-Lipschitz and the whole network is 1-Lipschitz.
4
Why does the Extreme Value Theorem fail for \(f(x) = e^{-x}\) on \(\mathbb{R}\)? How could you modify the problem to guarantee a minimum exists?
\(f(x) = e^{-x}\) is continuous and has infimum 0 on \(\mathbb{R}\), but \(f(x) > 0\) for all \(x\), so the infimum is never achieved. The EVT fails because \(\mathbb{R}\) is not compact (it is unbounded). Restricting the domain to any compact set like \([a, b]\) makes the minimum \(e^{-b}\) achieved at \(x = b\). In practice, this is equivalent to adding a box constraint to your optimisation problem.
5
To your colleague's question: for safety-critical systems, why is Lipschitz continuity preferable to mere continuity? What could go wrong with a continuous-but-not-Lipschitz function?
Continuity alone only says "small input changes produce small output changes" — but doesn't bound how small. A continuous function can have arbitrarily steep slopes, meaning a tiny adversarial perturbation could cause a huge output swing. Lipschitz continuity provides a quantitative bound: output change \(\leq L \times\) input change. For safety-critical systems, this means you can certify that input perturbations of size \(\varepsilon\) change outputs by at most \(L\varepsilon\) — a provable robustness guarantee. Without Lipschitz bounds, no such certificate is possible.
