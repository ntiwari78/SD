The Strategist's Toolkit
Nash Equilibrium
For all players $i$: $u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*)$
Minimax Theorem
$\max_p \min_q p^T A q = \min_q \max_p p^T A q$
Dominant Strategy
Strategy $s_i$ dominates if $u_i(s_i, s_{-i}) > u_i(s_i', s_{-i})$ for all $s_{-i}$
Shapley Value
$\phi_i(v) = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|!(n-|S|-1)!}{n!}[v(S \cup \{i\}) - v(S)]$
