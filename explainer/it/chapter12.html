Chapter 12

Information Theory and Statistics

The Bridge Between Data and Probability

In this chapter, we explore how information theory and statistics intertwine. We'll investigate the fundamental limits of statistical inference using the language of relative entropy, discover how empirical distributions behave under scrutiny, and learn universal compression schemes that work without knowing the source distribution.

The Two Jars Problem: A Feynman Perspective

"You have two jars of coins—one fair, one rigged. Flip a coin from an unknown jar n times. How quickly can you identify which jar? Sanov's theorem says the error drops exponentially at rate D(p||q)—the relative entropy between the two distributions."

This classic problem captures the essence of Chapter 12. With n i.i.d. samples from an unknown distribution, we want to decide whether it's fair (probability 1/2 for heads) or rigged (some other probability q). The speed at which we can make this decision—the exponential rate—is governed by the Kullback-Leibler divergence.

Why? Because the KL divergence D(p||q) measures how "far apart" two distributions are in the language that matters for exponential concentration: the probability of rare events.

Method of Types (Section 12.1)

The type of a sequence xn is the empirical distribution—the fraction of each symbol.

$$P_x(a) = \frac{\#\{i : x_i = a\}}{n}$$

For a binary alphabet, if we see k zeros and (n−k) ones, the type is simply k/n. Key facts about types:

  • Number of type classes: At most $(n+1)^{|\mathcal{X}|}$. For binary, just n+1 types.
  • Probability of a type: All sequences of the same type are roughly equally likely: $$\Pr(\text{type } P) \approx 2^{-nD(P||p)}$$ where p is the true source distribution.

Sequences with type P are about $2^{nH(P)}$ in count, and each has probability roughly $2^{-nH(P)}$ times a normalization factor.

Large Deviations & Sanov's Theorem (Sections 12.4-12.5)

Large Deviations Theory tells us how likely rare events are. For the empirical distribution $\hat{P}_n$:

$$\Pr(\hat{P}_n \in E) \approx 2^{-n \min_{q \in E} D(q||p)}$$

Sanov's Theorem: The probability that the empirical distribution falls into a set E decays exponentially, and the rate is the minimum KL divergence from p to any distribution in E. This is the fundamental theorem of large deviations for i.i.d. samples.

Interpretation: The empirical distribution concentrates around the true distribution p. Deviations are exponentially rare, with rare events weighted by their KL divergence from p.

  • Rate function: I(q) = D(q||p)
  • Intuition: Distributions "far" from p (in KL sense) are exponentially more unlikely.
  • Application: Bounds on hypothesis testing, compression, and channel coding.

Hypothesis Testing & Stein's Lemma (Section 12.7)

We observe n samples and must decide: Is the source distribution p (null hypothesis) or q (alternative)?

\begin{align} \alpha &= \Pr(\text{reject } p | p \text{ true}) \quad (\text{Type I error})\\ \beta &= \Pr(\text{accept } p | q \text{ true}) \quad (\text{Type II error}) \end{align}

Stein's Lemma: For a fixed Type I error α, the optimal Type II error exponent is: $$\lim_{n \to \infty} -\frac{1}{n}\log \beta^* = D(p||q)$$

This means the best we can do is have Type II error decay as $\beta^* \approx e^{-nD(p||q)}$. The KL divergence D(p||q) determines the difficulty of distinguishing p from q.

Chernoff Bound (Symmetric Testing): For testing p vs q equally, the error probability drops exponentially at rate D(p||q) at minimum.

Lempel-Ziv Coding (Section 12.10)

One of the most practical compression schemes. LZ77 and LZ78 adapt to any source distribution without knowing it in advance.

  • LZ77 (Sliding Window): Search for longest match in a sliding window of past data.
  • LZ78 (Dictionary): Build a dictionary of phrases as you parse the string.

Key Property: Both schemes achieve the entropy rate asymptotically, universally. No need to estimate the source distribution first.

Real-world use: These algorithms power gzip, ZIP, PNG, and many other compression formats.

$$\limsup_{n \to \infty} \frac{1}{n}L_n \leq H(X) \quad \text{almost surely}$$ where $L_n$ is the compressed length.

Fisher Information (Section 12.11)

The Fisher Information I(θ) measures how much information n i.i.d. samples carry about an unknown parameter θ:

$$I(\theta) = \mathbb{E}\left[\left(\frac{d}{d\theta}\log f(X;\theta)\right)^2\right] = -\mathbb{E}\left[\frac{d^2}{d\theta^2}\log f(X;\theta)\right]$$

Cramér-Rao Bound: For any unbiased estimator T of θ: $$\text{Var}(T) \geq \frac{1}{nI(\theta)}$$

The inverse Fisher Information gives a fundamental lower bound on estimator variance. Higher Fisher Information means we can estimate θ more precisely.

Connection to Information Theory: Under regularity conditions, Fisher Information relates to the second-order asymptotics of the KL divergence between nearby distributions.

Empirical Type & KL Divergence

import numpy as np
from scipy.special import comb
from collections import Counter

def empirical_type(sequence):
    """Compute empirical distribution (type) of a sequence."""
    counts = Counter(sequence)
    n = len(sequence)
    return {sym: counts[sym]/n for sym in counts}

def kl_divergence(p, q):
    """KL divergence D(p||q)."""
    div = 0
    for sym in p:
        if q.get(sym, 0) > 0:
            div += p[sym] * np.log2(p[sym] / q.get(sym, 1e-10))
    return div

# Example: two distributions
p = {0: 0.3, 1: 0.7}  # source
q = {0: 0.5, 1: 0.5}  # test distribution

d_kl = kl_divergence(p, q)
print(f"D(p||q) = {d_kl:.4f} bits")

LZ78 Compression

def lz78_compress(data):
    """LZ78 compression algorithm."""
    dictionary = {}
    w = ""
    output = []
    code = 0

    for c in data:
        wc = w + c
        if wc in dictionary:
            w = wc
        else:
            if w in dictionary:
                output.append((dictionary[w], c))
            else:
                output.append((0, c))
            dictionary[wc] = code
            code += 1
            w = ""

    if w:
        output.append((dictionary[w], ""))

    return output, dictionary

# Example: compress "ABRACADABRA"
compressed, _ = lz78_compress("ABRACADABRA")
print(f"Compressed: {compressed}")
print(f"Compression ratio: {len(compressed)} symbols → {len('ABRACADABRA')} chars")

Socratic Sandbox: Test Your Understanding

Reflect on these questions. Click "Show Hint" if you're stuck.

1. Why does Sanov's theorem say that the error exponent in hypothesis testing equals D(p||q)?

Think about the relative likelihoods of sequences under p vs q. What makes a sequence "rare" under one distribution?

Under the true distribution p, sequences with type P are roughly $2^{-nD(P||p)}$ times as likely as an "expected" sequence. Under q, their likelihood changes by a different exponent. The gap between these exponents gives the error rate.
2. In LZ78, why can we universally achieve entropy rate compression without knowing the source distribution?

How does the algorithm adapt to frequent vs. rare symbols?

LZ78 learns the distribution as it parses. Frequent phrases get shorter codes (small dictionary indices), while rare ones need larger codes. Over time, the average code length converges to H(X).
3. What is the geometric meaning of the KL divergence D(p||q) in the context of large deviations?

How does it measure "distance" between distributions for rare events?

D(p||q) is not a metric, but it quantifies how much "easier" it is to achieve a certain type under q vs p. A small KL means q assigns nearly as much probability to the type as p does; a large KL means q is much more "surprised" by the type, making it exponentially rarer under q.
4. How does the Cramér-Rao bound relate to Fisher Information, and why is it a lower bound?

Consider: what does it mean for an estimator to be "efficient"?

Fisher Information measures how "sensitive" the likelihood is to changes in θ. Higher sensitivity means we can pin down θ more precisely. The Cramér-Rao bound says no unbiased estimator can do better than 1/(nI(θ)) variance; estimators that achieve this are called efficient.

Summary

  • Types & Sanov's Theorem: The empirical distribution concentrates around the true distribution. Deviations are exponentially rare, with rate D(q||p).
  • Hypothesis Testing: The optimal Type II error exponent equals D(p||q), linking statistics to information theory.
  • Lempel-Ziv Compression: Universal algorithms achieve entropy rate without knowing the source distribution.
  • Fisher Information: Measures information in data; inverted, it gives the Cramér-Rao lower bound on estimator variance.
  • Unified Picture: Large deviations, hypothesis testing, compression, and estimation all pivot on KL divergence and entropy.

Chapter 12 closes the loop: information theory is not just about communication and compression. It is the language of statistics itself.
